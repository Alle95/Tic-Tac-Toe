Tic-Tac-Toe


X şi Zero este un joc la care participă doi jucători: unul joacă cu semnul X şi celălalt joacă cu 0. Ambii mută alternativ, începând cu jucătorul X. Ei pot muta doar în pătrăţelele libere. Iniţial, tabla este alcătuită din 9 pătrăţele goale, aliniate într-o tablă de 3x3. Scopul jocului este de a face 3 semne proprii pe o linie, coloană sau diagonală. Dacă tabla este plină dar nici un jucător nu a reuşit să îndeplinească acest scop, atunci jocul se termină remiză.
